First Game Jam

This was my first game jam done in 2023 for a game jam called "Jump Ship Jam." The theme of the game jam was "Strength in Numbers," which is why I made a horde-style survival game. This was my first game. I coded a full game, and I chose to use the Godot engine to start. I spent the first couple of days learning how to code in the engine before deciding on a theme. I decided on a space ship survival game because I had made a similar game in the past and wanted to recreate it. I mainly used a tutorial to get the enemies spawning and used free assets and music to get the game to look nice. It was a pretty fun experience and really got me into game development.

Second Game Jam

My second game jam was done a few months later for a Halloween-themed game jam. This time, I wanted to focus on a first-person 3d game. Because of that, I had more problems developing this game. I also joined this game jam pretty late, so I only had a few weeks or so to make the game. Getting used to 3d was much harder than 2d. I had to learn how to put textures on the assets, as well as learning how to spawn multiple levels. Overall, the experience was much more stressful, but I was pretty happy with the result, and I feel like it is important to at least try a 3d game out.
